## Funcionamiento:  
- **Ventana - Menu Principal**
    - _En la ventana del menu se presentan tres botones, dos de ellos con las opciones para realizar analisis de datos sobre diferentes
    temas y un tercero para finalizar el programa.
    Una vez presionada culquiera de las dos opciones(FIFA o VIDEOJUEGOS) se generara un archivo json a partir de un archivo csv(datasets)._  

- **Ventana - Analisis FIFA**
    - _Una vez seleccionada la opcion 'FIFA 19' se nos presentara una ventana con diferentes opciones para obtener 'x' datos,una vez presionado cualquiera de los botones generara un archivo '.json' y el mismo sera almacenado en el directorio ../data/json.
    A su vez se vera reflejado en una nueva ventana.  

- **Ventana - Analisis VIDEOJUEGOS**
    - _Una vez seleccionada la opcion 'VIDEOJUEGOS' se nos presentara una ventana con diferentes opciones para obtener 'x' datos,una vez presionado cualquiera de los botones generara un archivo '.json' y el mismo sera almacenado en el directorio ../data/json.
    A su vez se vera reflejado en una nueva ventana.
